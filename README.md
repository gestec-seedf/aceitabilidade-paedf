# Aceitabilidade · PAE-DF

App de bolso (PWA) para consulta rápida do **Boletim Aceitabilidade — 24ª edição (06/2026)** do Programa de Alimentação Escolar do Distrito Federal.

## Recursos

- 10 seções: Conceito, Base Legal, Escala Hedônica, Quando Aplicar, Resto-Ingestão, Adesão, Dentro-Fora, Glossário, Árvore Decisória e Fluxo PAE-DF.
- Busca em tempo real.
- 4 calculadoras: aceitação (escala hedônica), resto-ingestão, índice de adesão e tamanho da amostra (estratificada por Programa de Trabalho).
- Modo claro / escuro.
- Instalável como aplicativo no celular (PWA) e com funcionamento **offline**.

## Como usar

Abra a versão hospedada e, no celular, escolha **"Adicionar à tela inicial"** (iOS) ou **"Instalar aplicativo"** (Android).

## Fonte

Boletim Alimentação — 24ª edição · 06/2026 — Gerência de Supervisão Técnica e Educação Alimentar e Nutricional — DIAE/SEE-DF.
